🎯这个软件是干什么的？

想象一下：你正在写代码，需要一边看文档一边敲键盘。传统方法是什么？
不停地在代码编辑器和文档之间 Alt+Tab 切来切去，是不是很烦？😤

有了这个软件，你可以把文档悬浮在屏幕上，背景完全透明！
就像把笔记本贴在显示器上一样，既能看到代码，又能看到文档！👍

适合这些场景：

  • 编程时查 API 文档 - 代码和文档同时看，不用来回切换
  • 学习时记笔记 - 一边看视频一边显示重点笔记
  • 开会时看提纲 - 演示PPT的同时，悄悄看会议要点
  • 写作时参考资料 - 素材和编辑器同时显示

🚀新手快速入门（3分钟学会）

1

启动软件

最简单：双击 md_reader.exe，软件会自动打开示例文档

打开自己的文件：把你的 .md 或 .txt 文件拖到程序图标上

第一次使用不知道怎么开始？
就直接双击 md_reader.exe，会自动打开一个示例文档，你可以看看效果！
2

调整到舒服的状态

移动位置：按住鼠标左键拖动窗口到你想要的位置（比如屏幕右边）

让背景透明：

  1. 看屏幕右下角（时钟旁边），找到一个小图标（显示字母 M）
  2. 右键点击这个图标
  3. 选择 "背景透明度…"
  4. 把滑块拖到最左边（数值为 0），背景就完全透明了！
文字看不清怎么办？
如果背景是深色（比如黑色代码编辑器），文字颜色要选白色或黄色
如果背景是浅色（比如白色网页），文字颜色要选黑色或深蓝色
改颜色：右键托盘图标 → "文字颜色…" → 选一个你喜欢的颜色
3

开始阅读和翻页

翻页很简单，有3种方法：

操作 效果
鼠标滚轮上下滚动 一行一行慢慢看
右键点击窗口上半部分 快速往上翻半页 ↑（会显示箭头动画）
右键点击窗口下半部分 快速往下翻半页 ↓（会显示箭头动画）
什么是"半页"？
就是翻动半个窗口那么多的内容。比如窗口能显示20行，翻半页就是一次翻10行。
这样翻页既快速，又不会漏看内容！

🔒超级功能：锁定/穿透模式（重点！）

这是这个软件最强大的功能！理解了这个，你就真正会用这个软件了。

什么是"穿透模式"？用生活中的例子解释

想象一下：
你在电脑屏幕上贴了一张透明的便利贴（就是这个阅读窗口）。

🔓 解锁模式（交互模式）：
就像普通便利贴，你可以：
• 用鼠标拖动它
• 点击上面的按钮
• 在上面右键翻页
但是！这样的话，便利贴会挡住下面的窗口，你点不到下面的程序了。

🔒 锁定模式（穿透模式）：
就像这张便利贴变成了全息投影！虽然你能看到文字，但：
• 你的鼠标左键会穿透它，点到下面的窗口
• 你可以一边看文档，一边操作下面的程序
• 文档就像漂浮的幻影一样，完全不挡事！

这就是"穿透"的意思！

🔓 解锁模式（交互模式）

什么时候用：

  • 需要移动窗口位置
  • 需要打开新文件
  • 需要调整设置

特点：窗口正常工作，但会挡住下面的内容

🔒 锁定模式（穿透模式）

什么时候用：

  • 只是阅读，不需要操作窗口
  • 需要点击窗口下面的东西
  • 不想窗口挡住其他程序

特点：左键穿透，但右键翻页仍然有效！

如何切换这两种模式？（3种方法）

方法1：点击锁定图标（最简单）

看窗口右上角，关闭按钮 [✕] 的左边，有一个锁形图标：

  • 🔓 开锁图标 = 解锁模式（可以拖动窗口）
  • 🔒 闭锁图标 = 锁定模式（鼠标会穿透）

点击这个图标就能切换模式！鼠标悬停时图标会高亮提示。

方法2：用快捷键（高手都用这个）

按 Ctrl + Shift + L

记忆方法：L = Lock（锁定）

方法3：托盘菜单

右键点击右下角的托盘图标 → 选择 "🔒 锁定窗口（穿透模式）"

这个选项前面有勾✓ = 已锁定，没勾 = 未锁定

🎯 临时需要拖动怎么办？

比如你开启了锁定模式（窗口穿透），突然想临时移动一下窗口位置，
不需要关闭锁定！只要：

按住 Ctrl 键，然后拖动窗口
松开 Ctrl，窗口又变回穿透模式了！

就像临时"抓住"了那张透明便利贴，移动完又让它飘回去！
重要提示：锁定模式下，右键翻页仍然有效！
即使窗口处于穿透模式，你仍然可以：
• 右键窗口上半部分 → 向上翻页 ↑
• 右键窗口下半部分 → 向下翻页 ↓
• 点击关闭按钮 [✕] → 隐藏窗口
• 点击锁定图标 🔒 → 切换模式

📚智能历史记录功能（超强！）

这个软件会用 SQLite 数据库为每个文件独立保存阅读历史！
就像浏览器会记住每个网页的浏览位置一样，这个软件会记住每个文档的阅读进度。

会保存什么？

每个文件独立保存：

  • 📍 阅读进度：看到第几行了
  • 🔒 锁定状态：这个文件是否开启穿透模式

全局设置（所有文件共享）：

  • 🎨 文字颜色
  • 📊 背景透明度
  • 📝 文字透明度
  • 📄 最后打开的文件

实际使用场景举例

1

你打开 Python教程.md，看到第50行，保持解锁模式（因为要操作窗口）

→ 软件记录：Python教程.md = 第50行 + 解锁

2

切换到 API文档.md，看到第80行，开启锁定模式（方便边看边写代码）

→ 软件记录：API文档.md = 第80行 + 锁定

3

关闭软件，一周后重新打开...

4

拖入 Python教程.md

→ ✨ 自动跳到第50行，自动设为解锁模式

5

拖入 API文档.md

→ ✨ 自动跳到第80行，自动设为锁定模式

就像每本书都有独立的书签！
不管你看了多少个文档，软件都能精确记住每一个的阅读进度和你的使用习惯。
下次打开时，一切都恢复到上次离开的状态，无需手动设置！

数据存储位置

软件会在同目录自动创建：

  • history/history.db - SQLite数据库，存储所有文件的阅读历史
  • md_reader_config.ini - 配置文件，存储全局样式设置
想清除历史记录？
删除 history 文件夹和 md_reader_config.ini 文件即可。
下次启动会重新创建，所有设置恢复为默认值。

Ctrl + Alt + H 隐藏/显示窗口 临时不想看文档时

记不住快捷键？
没关系！所有功能都可以用鼠标操作，快捷键只是为了更方便。
用得多了，自然就记住了！

🎛️托盘图标怎么用？

托盘图标在哪里？

看屏幕右下角，时钟旁边，有很多小图标对吧？
找到一个显示字母 M 的图标，那就是我们的软件！

找不到图标？
可能被折叠了！点击右下角的小箭头 ▲，展开隐藏的图标。

对这个图标能做什么？

操作 会发生什么
左键点击 显示/隐藏阅读窗口（快速切换）
右键点击 打开功能菜单（所有设置都在这里）

右键菜单里有什么？

📊 背景透明度…

控制窗口背景有多透明（0 = 完全透明，255 = 完全不透明）

建议：设置为 0，这样背景完全透明，不挡东西

📝 文字透明度…

控制文字有多透明（0 = 看不见，255 = 完全清晰）

建议：设置为 200-255，文字清晰好看

🎨 文字颜色…

打开颜色选择器，选一个你喜欢的文字颜色

可以实时预览！移动色盘，窗口文字会立刻变颜色

满意就点"确定"，不满意就点"取消"恢复原样

🔒 锁定窗口（穿透模式）

开关穿透模式（前面打勾✓ = 已锁定）

点一下就能切换锁定/解锁状态

🔄 重新加载当前文件

如果你在其他软件里修改了正在看的文档，点这个刷新内容

📂 打开文件…

弹出文件选择窗口，选一个新的文档来看

❓常见问题解答

Q1: 窗口不见了！怎么找回来？

A: 很可能是被隐藏了，不是关闭了：

  • 方法1：左键点击右下角托盘图标（字母 M 那个）
  • 方法2：按快捷键 Ctrl + Alt + H

Q2: 我开启了锁定模式，为什么鼠标点不到窗口了？

A: 这是正常的！锁定模式就是让鼠标穿透的。

  • 如果想关闭锁定：按住 Ctrl，然后点击右上角的 🔒 图标
  • 或者：按快捷键 Ctrl + Shift + L

Q3: 文字看不清，怎么调整？

A: 试试这些方法：

  1. 调高文字透明度：右键托盘 → "文字透明度…" → 拖到 255
  2. 换个对比度高的颜色：右键托盘 → "文字颜色…" → 选白色或黄色
  3. 调低背景透明度：右键托盘 → "背景透明度…" → 调到 20-50（给文字加个淡背景）

Q4: 中文显示乱码？

A: 文件编码问题。解决方法：

  1. 用记事本打开你的文件
  2. 点"文件" → "另存为"
  3. 在底部"编码"下拉框选择 UTF-8
  4. 保存，再用本软件打开

Q5: 怎么彻底关闭软件？

A: 点关闭按钮 [✕] 只是隐藏窗口，软件还在后台运行。

真正退出的方法：

  • 方法1：右键托盘图标 → 选择"退出"
  • 方法2：按快捷键 Ctrl + Alt + Esc

Q6: 为什么提示"程序已经在运行中"？

A: 软件已经启动了，只能运行一个。

解决方法：

  • 检查右下角托盘图标，如果有就左键点击显示窗口
  • 如果要完全重启，先右键托盘图标 → "退出"，再重新打开软件

Q7: 软件会记住我的设置吗？

A: 会！软件会自动保存：

  • 上次打开的文件
  • 阅读进度（滚动到哪一行了）
  • 文字颜色和透明度
  • 背景透明度
  • 锁定状态（是否穿透）

下次启动会自动恢复，不用重新设置！

💡高级使用技巧

技巧1：设为默认打开程序

让 .md 文件双击就用这个软件打开：

  1. 在文件资源管理器，找到一个 .md 文件
  2. 右键点击 → "打开方式" → "选择其他应用"
  3. 点"更多应用" → "在这台电脑上查找其他应用"
  4. 找到 md_reader.exe 并选择
  5. 勾选"始终使用此应用打开 .md 文件"
  6. 点"确定"

以后双击 .md 文件，就自动用这个软件打开了！

技巧2：编程时的最佳设置

  • 位置：窗口放在屏幕右边或左边，不要挡住代码区
  • 背景透明度：0（完全透明）
  • 文字颜色：如果代码编辑器是深色主题，用白色或青色文字
  • 模式：阅读时开启锁定模式（穿透），这样不影响写代码
  • 快捷操作：用右键快速翻页，比滚轮快！

技巧3：多显示器使用

如果你有两个显示器：

  • 主显示器：工作区（写代码、写文档）
  • 副显示器：把阅读窗口放这里，设置为解锁模式，方便随时操作

这样可以充分利用屏幕空间！

技巧4：会议演示时悄悄看提纲

  1. 把会议要点写成 .md 文件
  2. 用本软件打开，窗口放在PPT下方
  3. 背景透明度设为 0
  4. 开启锁定模式（穿透），这样点击PPT不会被窗口挡住
  5. 需要翻页时，右键点击窗口

观众看不出来你在看提纲！😎

💻系统要求和其他信息

系统要求

  • 操作系统：Windows 7 / 8 / 10 / 11
  • 运行库：不需要！解压就能用
  • 硬盘空间：约 300KB（非常小）
  • 支持格式：.md 和 .txt 文件（UTF-8 编码）

配置文件

软件会在同目录创建 md_reader_config.ini，保存你的设置。

想重置所有设置？删除这个文件，重新打开软件即可。

隐私和安全

  • ✅ 完全本地运行，不联网
  • ✅ 不收集任何信息
  • ✅ 不修改系统文件
  • ✅ 配置保存在程序目录，卸载时直接删除文件夹即可
  • ✅ 开源项目，代码可审查
